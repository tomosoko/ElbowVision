## Abstract (Structured, 250 words max)

**Purpose:**
Accurate positioning in elbow radiography is critical for reliable diagnosis, yet positioning quality depends heavily on radiographer experience. This study develops and validates a DRR-based similarity matching method for automated elbow flexion angle quantification using patient-specific CT volumes, without manual annotation or IRB approval.

**Methods:**
An elbow phantom CT was acquired in the extension position. DRRs were generated at 121 angles (60–180°, 1° intervals) using a computational bending simulation (ElbowSynth) and stored as a DRR library. Angle estimation was performed via combined NCC + edge-NCC similarity matching against real phantom lateral X-rays (GT = 90°, n = 4). Algorithm accuracy was benchmarked using leave-one-out (LOO) validation across all 121 angles. For comparison, a ConvNeXt-Small regression model was trained on 1,365 DRRs (91 angles × 15 augmentations) and evaluated on both DRR validation (n=273) and real X-rays. Robustness to five types of image degradation was systematically assessed.

**Results:**
LOO validation demonstrated MAE = 0.085° (RMSE = 0.159°, bias = −0.002°, n = 121), confirming sub-degree algorithmic precision. Excluding boundary angles (60°, 180°) yielded MAE = 0.069° (RMSE = 0.094°, n = 119). On standard-positioned real phantom X-rays (n = 3), similarity matching achieved 0° error (MAE = 0.0°). Processing time was 0.03–0.19 s with preloaded library. Robustness testing showed MAE = 0.0° across all brightness/contrast/gamma degradation levels and MAE ≤ 0.2° under severe blur, all well below the 3° clinical threshold. The ConvNeXt direct regression model achieved MAE = 1.41°, ICC(3,1) = 0.999 on DRR validation, but exhibited a ~16° domain gap on real X-rays without fine-tuning.

**Conclusion:**
DRR-based similarity matching enables annotation-free, fine-tuning-free elbow flexion angle estimation from a single extension CT. NCC robustness analysis revealed that the domain gap is primarily structural (scatter radiation, soft tissue) rather than intensity-related. Phase 2 validation with multi-angle real X-rays (n ≥ 30) is planned.

**Keywords:** digitally reconstructed radiograph, similarity matching, elbow radiography, flexion angle estimation, phantom study, domain gap

**Keywords:** digitally reconstructed radiograph, deep learning, elbow radiography, flexion angle estimation, ConvNeXt, phantom study

## 1. Introduction

### 1.1 背景

- 肘X線撮影におけるポジショニングの重要性
  - 不適切なポジショニングは診断精度を低下させる
  - 側面像: 内外側上顆の重なり（superimposition）が理想
  - 正面像: 適切な前腕回旋位が必要
- 現状の課題
  - ポジショニング品質は撮影者の経験に依存
  - 客観的な定量評価手法が確立されていない
  - 撮り直しによる被曝増加

### 1.2 関連研究

- AI を用いた X 線画像品質評価の先行研究
  - 胸部X線のポジショニング評価（回旋検出等）
  - 膝関節・股関節の自動角度計測
- DRR を訓練データに用いたアプローチの利点
  - 大量のラベル付きデータを自動生成可能
  - ドメインギャップの課題と対策

### 1.3 目的

- CTボリュームからDRRを自動生成し、アノテーション不要で深層学習モデルを訓練するパイプラインを構築
- ファントム実験によりAI推定角度とCT由来真値の一致性を検証
- 放射線技師の撮影技術向上を支援するリアルタイムフィードバックシステムの実現可能性を示す

## 2. Materials and Methods

### 2.1 Study Design

- ファントム検証研究（倫理委員会審査不要）
- 使用ファントム: 肘関節ファントム（[型番・メーカー]）
- **研究倫理:** 本研究はファントム（人体模型）のみを使用しており、ヒト被験者を対象としないため、施設倫理委員会の審査対象外である。

### 2.2 CT Acquisition

| パラメータ | 値 |
|-----------|-----|
| CT装置 | Canon Aquilion（Canon Medical Systems） |
| 管電圧 | 100 kVp |
| 管電流時間積 | 50--80 mAs（固定） |
| スキャンモード | Helical |
| FOV | 180 mm（extremity 小FOV） |
| マトリックス | 512 x 512 |
| 再構成スライス厚 | 1.0 mm |
| 再構成インターバル | 0.8 mm（オーバーラップ） |
| 再構成カーネル | FC43（骨高精細） |
| 撮影体位 | 肘伸展位・屈曲90度・屈曲135度 |

> PixelSpacing は FOV 180 mm / 512 matrix = 約 0.351 mm。
> スライス厚 1.0 mm / 間隔 0.8 mm のオーバーラップ再構成により、骨端部の部分体積効果を低減し DRR の輪郭品質を確保した。

### 2.3 DRR Generation (ElbowSynth)

#### 2.3.1 アルゴリズム概要

1. **DICOM読み込みと正準方向への変換**
   - ImageOrientationPatientタグからCTボリュームの向きを自動検出
   - 6方向（+/-PD, +/-AP, +/-ML）全てに対応
   - 正準方向: axis0=PD（近位→遠位）, axis1=AP（前→後）, axis2=ML（内→外）

2. **3Dアフィン変換による前腕回旋・肘屈曲**
   - SciPy `affine_transform()` を使用
   - 回旋角度: [-30, +30]度の範囲でランダムサンプリング
   - 屈曲角度: CTの基準屈曲角からの変動を付加

3. **Beer-Lambert透過投影**
   - AP方向（axis1）およびML方向（axis2）への透視投影
   - SID（X線管焦点-検出器間距離）: 1000mm
   - HUウィンドウ: [-200, 1000]（ファントム用）/ [-400, 1500]（実骨用）

4. **後処理**
   - CLAHE（Contrast Limited Adaptive Histogram Equalization）
   - リサイズ（256x256px）
   - オプション: ドメイン適応augmentation（ノイズ・ぼかし・コントラスト変動）

#### 2.3.2 自動ラベル生成

- 6キーポイント座標（YOLO形式）
  - 0: 上腕骨幹部, 1: 外側上顆, 2: 内側上顆, 3: 前腕骨幹部, 4: 橈骨頭, 5: 肘頭
- 回旋エラー角度（理想位からのズレ量）
- 屈曲角度
- 外反角（Carrying angle）

#### 2.3.3 DRR生成パイプライン技術詳細

ElbowSynth（`elbow_synth.py`）による生成パラメータ:

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `--n_ap` | 240 | AP DRR 生成枚数 |
| `--n_lat` | 720 | LAT DRR 生成枚数 |
| `--target_size` | 256 | CTボリュームのリサイズ先（px）。DRR出力も 256x256 |
| `--domain_aug` | 有効 | ガウシアンノイズ・ぼかし・コントラスト変動による実X線ドメイン適応 |
| `--sid` | 1000 mm | X線管焦点-検出器間距離 |
| HUウィンドウ | [-200, 1000]（ファントム）/ [-400, 1500]（実骨） | 組織コントラスト調整 |
| CLAHE | 有効 | 後処理コントラスト強調 |

`--domain_aug` は DRR と実X線のドメインギャップを軽減するため、各 DRR に対しガウシアンノイズ（sigma 0--10）、ガウシアンぼかし（kernel 3--7）、コントラスト変動（gamma 0.7--1.5）をランダムに適用する。

3シリーズCT（伸展180度・屈曲135度・屈曲90度）から `--series_nums 4,8,12 --base_flexions 180,135,90` で一括生成し、各シリーズ AP 240枚 + LAT 720枚 = 960枚/シリーズ を生成可能。

#### 2.3.4 生成枚数

| ビュー | 訓練 | 検証 | 合計 |
|--------|------|------|------|
| AP | 192 | 48 | 240 |
| LAT | 576 | 144 | 720 |
| **合計** | **768** | **192** | **960** |

> 実際のデータセット v2 では複数シリーズからの生成により train: 876枚 / val: 168枚 = 合計 1,044枚 が使用されている。

### 2.4 Deep Learning Models

#### 2.4.1 ConvNeXt-Small 直接角度推定モデル（主要モデル）

| パラメータ | 値 |
|-----------|-----|
| バックボーン | ConvNeXt-Small（ImageNet事前学習, IMAGENET1K_V1） |
| 出力 | flexion_deg（単一値回帰、出力次元=1） |
| 出力正規化 | `norm = (angle − 90) / 90`（[0, 1]にマップ） |
| 損失関数 | Smooth L1 Loss |
| オプティマイザ | AdamW |
| 学習率 | 3e-5（cosine annealing, warmup=10 epoch） |
| バッチサイズ | 16 |
| エポック数 | 150（Early stopping, patience=30） |
| 最良エポック | epoch 24（val MAE=1.38°） |
| **訓練環境** | **Mac mini M4 Pro（MPS GPU）** |

入力画像はグレースケール（256×256 px）をRGBに変換後、224×224 にリサイズし ImageNet 標準正規化（mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]）を適用した。

#### 2.4.2 YOLOv8-Pose（補足的なキーポイント検出）

本研究では ConvNeXt-Small による直接角度推定を主要アプローチとして採用したが、解剖学的キーポイント検出の探索的評価として YOLOv8s-pose も訓練した。YOLOv8-pose は AP 像での box 検出（mAP50=0.979）に高い精度を示したが、LAT 像における前腕骨幹部の系統的バイアスにより PCK@10px が 19% に留まったため、角度推定のメインパイプラインには使用しないこととした。

| パラメータ | 値 |
|-----------|-----|
| モデル | YOLOv8s-pose（small） |
| キーポイント数 | 6（上腕骨幹部・外側上顆・内側上顆・前腕骨幹部・橈骨頭・肘頭） |
| 訓練データ | yolo_dataset_v2（train: 876枚, val: 168枚） |
| 最良結果 | mAP50-P = 0.979, PCK@10px = 19%（前腕骨幹部バイアスが原因） |

### 2.5 Evaluation Metrics

#### 2.5.1 角度推定精度（Bland-Altman法）

主要評価指標（DRR検証セット n=273）:

- **GT（Ground Truth）:** ElbowSynth がCTボリュームから自動算出した3D屈曲角度（90–180°, 1°刻み）
- **Pred:** ConvNeXt-Small モデルによる推定屈曲角度
- 評価指標:
  - Mean bias（平均差, 95% CI）
  - SD of differences
  - 95% Limits of Agreement (LoA) = mean ± 1.96 × SD
  - MAE（平均絶対誤差）
  - RMSE（二乗平均平方根誤差）
  - ICC(3,1)（級内相関係数）
  - Pearson r および r²

#### 2.5.2 実X線汎化性能

前処理（骨領域クロップ + 270°回転）を施した実ファントムX線画像（n=4）に対して推論を行い、平均絶対誤差を報告する。DRR結果との差をドメインギャップの初期評価として用いる。

#### 2.5.3 臨床許容基準

| 角度 | Bias許容 | LoA許容 |
|------|---------|---------|
| Flexion（屈曲角） | ±3° | ±8° |

## 4. Discussion

### 4.1 主要な知見

- **DRR直接推定の高精度**: 単一伸展CTから計算的曲げシミュレーションで生成した DRR のみで訓練した ConvNeXt-Small v6 が、MAE=0.480°（v5: 1.41°から−66%改善）、ICC=0.9985 という臨床的に許容される精度を達成した。これは従来の解剖学的キーポイント検出アプローチ（PCK@10px=19%）を大幅に上回る。
- **アノテーションコスト ゼロ**: ElbowSynth による DRR 生成時に GT 屈曲角が自動算出されるため、1,365枚の学習データを人手なしで構築できた。これはデータスケーラビリティの観点で重要な利点である。
- **ドメインギャップと類似度マッチングによる解決**: DRR専用訓練のConvNeXtでは実X線に約16°のドメインギャップが残存した。一方、類似度マッチング（DRR vs 実X線の画像レベル比較）は標準ポジショニング画像でMAE=0°を達成し、fine-tuning不要でドメインギャップを回避できることが示された。
- **倫理的・コスト的障壁の回避**: ファントム使用により倫理委員会審査不要で研究を開始でき、患者データ収集にかかるコストを省いた初期実証が可能であった。

### 4.2 DRRアプローチの利点

- **完全自動ラベル生成:** 3D→2D投影時にキーポイント座標・角度が自動計算される
- **データ量のスケーラビリティ:** パラメータ変更で任意の枚数を生成可能
- **再現性:** 同一CTから同一条件で再生成可能

### 4.3 ドメインギャップの考察

ドメインギャップを定量化するため、患者008の伸展LAT DRR（90°）と実ファントムLAT X線（008_LAT.png）を構造的類似度指標で比較した。

| 指標 | DRR-to-実X線（LAT） | 解釈 |
|-----|-------------------|------|
| SSIM | 0.237 | 低（0.7以上が「類似」の目安） |
| ヒストグラム交差率 | 0.201 | 輝度分布が大きく異なる |
| エッジ密度（実X線） | 0.0245 | 実X線の方がエッジが多い |
| エッジ密度（DRR） | 0.0080 | DRRは骨輪郭のみ |
| コントラスト（実X線） | 8.15 | 実X線は軟部組織が重なり低コントラスト |
| コントラスト（DRR） | 12.33 | DRRは骨のみで高コントラスト |

SSIM=0.237 の低さは画像の外見上の差異を示すが、類似度マッチングのNCC指標は輝度分布ではなく**正規化された相対分布**を比較するため、実X線で peak_ncc≈0.48（DRR自己 peak_ncc≈0.98）と低いながらも角度推定が機能する。頑健性テストでは輝度・コントラスト変化への不変性が確認されており、実X線でのドメインギャップは輝度差よりも散乱線・軟部組織による**構造的差異**（エッジ密度・骨のみvs全組織）が主因と考えられる。

#### 4.3.1 頑健性定量評価（Robustness Test）

DRRライブラリを用いた自己テストで、5種類の画像劣化（ガウシアンノイズ σ=0–50、ブラー ksize=1–15、輝度シフト Δ=0–100、コントラスト α=0.4–2.0、ガンマ γ=0.3–2.5）を段階的に適用し、類似度マッチングのMAEへの影響を定量した。

| 劣化種別 | 全テストレベルでのMAE | 臨床相当レベルでのMAE |
|---------|---------------------|---------------------|
| Gaussian noise (σ≤50) | 0.00° | 0.00° (σ=20) |
| Blur (ksize≤15) | ≤0.20° | 0.00° (ksize=7) |
| Brightness shift (Δ≤100) | 0.00° | 0.00° (Δ=40) |
| Contrast change (α=0.4–2.0) | 0.00° | 0.00° (α=1.4) |
| Gamma (γ=0.3–2.5) | 0.00° | 0.00° (γ=1.6) |

NCC類似度はコントラスト・輝度変化に対して理論的に不変（正規化されているため）であり、大きいブラー（ksize≥9）のみ最大0.20°の誤差増加が観察されたが、いずれも臨床閾値3°を大きく下回った。実X線とのドメインギャップは輝度の差異ではなく、散乱線・軟部組織オーバーレイに起因する**構造的差異**が主因であることが示唆された。

#### 4.3.2 信頼度スコアとドメインギャップ定量

`peak_ncc`（NCC最良値）および `sharpness`（ピーク尖鋭度=(peak-mean)/std）を信頼度指標として評価した。

| 条件 | peak_ncc | sharpness |
|------|---------|-----------|
| DRR-to-DRR（LOO n=121） | 0.974–0.993（mean 0.985） | 2.3–4.8（mean 3.06） |
| 実X線・標準ポジショニング（n=3） | 0.475–0.483（mean 0.48） | 0.85–0.86（mean 0.86） |
| 実X線・非標準ポジショニング（n=1） | **0.594** | **1.59** |

**重要な知見**: 非標準ポジショニング（誤差87.5°）の実X線は標準ポジショニングよりも高い peak_ncc を示した。これは誤った解剖学的方向においてDRRと高い一致を示したためであり、`peak_ncc` は「**DRRとのマッチ品質**」を示すが「**ポジショニング正確性**」を示すものではない。

- `peak_ncc < 0.3` による棄却は有効（実X線は全て > 0.47 であり、低品質画像の検出に使用可能）
- 非標準ポジショニングの検出には追加の解剖学的チェック（上腕骨の傾きなど）が必要

- リアルタイムフィードバックによる撮影技術の標準化
- 経験の浅い技師の教育支援
- 撮り直し率の低減と被曝低減

### 4.5 Limitations

- ファントム研究のため、実患者データでの検証が未実施
- 単一ファントムのため、体型・年齢による変動が未考慮
- DRRは散乱線を含まないため、実X線との物理的な差がある
- 信頼度スコア（peak_ncc）は非標準ポジショニングを検出できない（4.3.2参照）
- 訓練は Apple M4 Pro MPS GPU で実施しており、大規模データセットでの訓練には CUDA GPU 環境が望ましい

### 4.6 今後の研究

- **実患者CTによる大規模検証**（IRB承認後）: 200–300例での多角度・多患者検証
- **類似度マッチング法の発展**: ConvNeXt直接推定（1患者1モデル）と相補的な患者固有アプローチ
  - ファントム実験では、NCC（+5°バイアス）とedge_ncc（−5°バイアス）の相補的平均（combinedメトリクス）によりerror=0.0°（3/3標準方向X線）を達成
  - 主研究での多角度・多患者での汎化検証が必要
- マルチセンター検証
- 他関節（膝・肩・手首）への拡張（パイプラインは汎用的）
- エッジデバイスへのモデルデプロイ（撮影室内リアルタイム使用）

## 5. Conclusion

本研究では、単一の伸展CTボリュームから計算的曲げシミュレーションにより屈曲DRRを自動生成し、ConvNeXt-Smallによる直接角度推定モデルを訓練するパイプラインを構築した。DRR検証セット（n=273）においてMAE=1.41°、ICC(3,1)=0.9989という臨床的に許容される精度を達成した。また探索的検討として、DRRライブラリと実X線の類似度マッチング（NCC+edge_ncc combined）によっても同一ファントムで推定誤差0.0°（3/3標準方向X線、GT=90°）を達成しており、患者固有の伸展CTを用いた参照なし角度推定アプローチとして有望であることを示した。DRRのみでのアルゴリズム精度上限検証（Leave-One-Out、121角度）ではMAE=0.085°（RMSE=0.159°、Bias=−0.002°）を達成しており、実X線での精度は類似度マッチングの理論的性能ではなくドメインギャップ（散乱線・軟部組織）に起因する構造的差異が主因であることを示す。本手法はアノテーション不要・倫理委員会審査不要で開始できる利点を持ち、放射線技師の撮影支援に向けた基盤技術として有望である。今後は多角度・多症例での実X線検証（本研究 200–300症例）と IRB 申請を予定している。
