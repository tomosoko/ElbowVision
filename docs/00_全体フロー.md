# ElbowVision 全体ワークフロー

## プロジェクト概要

肘ファントムCTから DRR（模擬X線）を生成し、YOLOv8-Pose でキーポイント検出 → 関節角度を推定するシステム。
アノテーション不要：CT→DRRの際にキーポイント座標を自動生成する。
精度検証は同じファントムの実X線（レントゲン）を使う。

---

## 全体の流れ

```
Step 1  環境構築
        │  Python / npm install
        ▼
Step 2  ファントムCT撮影（職場）
        │  Canon CT（1mm スライス）でファントムを撮影
        │  → data/raw_dicom/ct/CT_001/ に DICOM を配置
        ▼
Step 3  DRR生成 + YOLOラベル自動生成（アノテーション不要）
        │  make factory
        │  yolo_pose_factory.py が実行される:
        │    - ct_reorient.py で解剖学的座標系に自動補正
        │    - AP/LAT の DRR を屈曲・回内外バリエーションで一括生成
        │    - 3D→2D投影でキーポイント座標を自動計算
        │    - YOLOラベル (.txt) を自動出力
        │  → data/yolo_dataset/ に画像・ラベル・dataset.yaml が揃う
        ▼
Step 4  モデル訓練
        │  make train
        │  train_yolo_pose.py で YOLOv8n-pose を fine-tune
        │  → runs/pose/elbowvision_pose_model/weights/best.pt
        ▼
Step 5  システム起動
        │  make api（ポート8000）+ make frontend（ポート3000）
        ▼
Step 6  精度検証（実X線で確認）
        │  同じファントムを実X線（レントゲン）で撮影
        │  → dicom_to_png.py で PNG 変換
        │  → make analyze でAI自動計測 → CSV
        │  → 手動計測値と比較して Bland-Altman 解析
```
