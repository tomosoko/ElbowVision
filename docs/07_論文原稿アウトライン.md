## Abstract (Structured, 250 words max)

**Purpose:**
Accurate positioning in elbow radiography is critical for reliable diagnosis, yet positioning quality depends heavily on radiographer experience. This study proposes an automated elbow positioning assessment system that uses digitally reconstructed radiographs (DRRs) generated from CT volumes to train deep learning models, eliminating the need for manual annotation and enabling objective, real-time positioning feedback.

**Methods:**
An elbow phantom was scanned using multi-detector CT at three flexion positions (90°, 135°, 180°). DRRs were generated from the extension CT volume at 91 discrete flexion angles (90–180°, 1° intervals) using a Beer-Lambert projection algorithm (ElbowSynth) with computational bending simulation, producing 1,365 synthetic lateral X-ray images with automatically calculated CT-derived ground-truth angles (15 augmentations per angle). A ConvNeXt-Small regression model was trained on this dataset to directly estimate elbow flexion angle from a single X-ray image. Agreement between AI-estimated and CT-derived ground-truth angles was evaluated using Bland-Altman analysis, intraclass correlation coefficient (ICC), and mean absolute error (MAE) on a held-out validation set (n=273).

**Results:**
On the DRR validation set (n=273), the model achieved MAE of 1.41°, mean bias of −1.25° (95% CI: −1.40 to −1.10°), and 95% limits of agreement of −3.71 to +1.20°, with ICC(3,1) = 0.999 and r² = 0.996. Both clinical acceptance thresholds were met (bias ≤±3°, LoA ≤±8°). Similarity matching on standard-positioned real phantom X-ray images (n=3) achieved 0° error through a combined NCC+edge-NCC metric.

**Conclusion:**
A DRR-based training pipeline using computational bending simulation of a single extension CT enables fully automated generation of labeled training data, achieving clinically acceptable flexion angle estimation accuracy without manual annotation or ethical approval. This phantom study demonstrates the feasibility of DRR-trained deep learning for elbow X-ray angle quantification.

**Keywords:** digitally reconstructed radiograph, deep learning, elbow radiography, flexion angle estimation, ConvNeXt, phantom study

## 1. Introduction

### 1.1 背景

- 肘X線撮影におけるポジショニングの重要性
  - 不適切なポジショニングは診断精度を低下させる
  - 側面像: 内外側上顆の重なり（superimposition）が理想
  - 正面像: 適切な前腕回旋位が必要
- 現状の課題
  - ポジショニング品質は撮影者の経験に依存
  - 客観的な定量評価手法が確立されていない
  - 撮り直しによる被曝増加

### 1.2 関連研究

- AI を用いた X 線画像品質評価の先行研究
  - 胸部X線のポジショニング評価（回旋検出等）
  - 膝関節・股関節の自動角度計測
- DRR を訓練データに用いたアプローチの利点
  - 大量のラベル付きデータを自動生成可能
  - ドメインギャップの課題と対策

### 1.3 目的

- CTボリュームからDRRを自動生成し、アノテーション不要で深層学習モデルを訓練するパイプラインを構築
- ファントム実験によりAI推定角度とCT由来真値の一致性を検証
- 放射線技師の撮影技術向上を支援するリアルタイムフィードバックシステムの実現可能性を示す

## 2. Materials and Methods

### 2.1 Study Design

- ファントム検証研究（倫理委員会審査不要）
- 使用ファントム: 肘関節ファントム（[型番・メーカー]）
- **研究倫理:** 本研究はファントム（人体模型）のみを使用しており、ヒト被験者を対象としないため、施設倫理委員会の審査対象外である。

### 2.2 CT Acquisition

| パラメータ | 値 |
|-----------|-----|
| CT装置 | Canon Aquilion（Canon Medical Systems） |
| 管電圧 | 100 kVp |
| 管電流時間積 | 50--80 mAs（固定） |
| スキャンモード | Helical |
| FOV | 180 mm（extremity 小FOV） |
| マトリックス | 512 x 512 |
| 再構成スライス厚 | 1.0 mm |
| 再構成インターバル | 0.8 mm（オーバーラップ） |
| 再構成カーネル | FC43（骨高精細） |
| 撮影体位 | 肘伸展位・屈曲90度・屈曲135度 |

> PixelSpacing は FOV 180 mm / 512 matrix = 約 0.351 mm。
> スライス厚 1.0 mm / 間隔 0.8 mm のオーバーラップ再構成により、骨端部の部分体積効果を低減し DRR の輪郭品質を確保した。

### 2.3 DRR Generation (ElbowSynth)

#### 2.3.1 アルゴリズム概要

1. **DICOM読み込みと正準方向への変換**
   - ImageOrientationPatientタグからCTボリュームの向きを自動検出
   - 6方向（+/-PD, +/-AP, +/-ML）全てに対応
   - 正準方向: axis0=PD（近位→遠位）, axis1=AP（前→後）, axis2=ML（内→外）

2. **3Dアフィン変換による前腕回旋・肘屈曲**
   - SciPy `affine_transform()` を使用
   - 回旋角度: [-30, +30]度の範囲でランダムサンプリング
   - 屈曲角度: CTの基準屈曲角からの変動を付加

3. **Beer-Lambert透過投影**
   - AP方向（axis1）およびML方向（axis2）への透視投影
   - SID（X線管焦点-検出器間距離）: 1000mm
   - HUウィンドウ: [-200, 1000]（ファントム用）/ [-400, 1500]（実骨用）

4. **後処理**
   - CLAHE（Contrast Limited Adaptive Histogram Equalization）
   - リサイズ（256x256px）
   - オプション: ドメイン適応augmentation（ノイズ・ぼかし・コントラスト変動）

#### 2.3.2 自動ラベル生成

- 6キーポイント座標（YOLO形式）
  - 0: 上腕骨幹部, 1: 外側上顆, 2: 内側上顆, 3: 前腕骨幹部, 4: 橈骨頭, 5: 肘頭
- 回旋エラー角度（理想位からのズレ量）
- 屈曲角度
- 外反角（Carrying angle）

#### 2.3.3 DRR生成パイプライン技術詳細

ElbowSynth（`elbow_synth.py`）による生成パラメータ:

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `--n_ap` | 240 | AP DRR 生成枚数 |
| `--n_lat` | 720 | LAT DRR 生成枚数 |
| `--target_size` | 256 | CTボリュームのリサイズ先（px）。DRR出力も 256x256 |
| `--domain_aug` | 有効 | ガウシアンノイズ・ぼかし・コントラスト変動による実X線ドメイン適応 |
| `--sid` | 1000 mm | X線管焦点-検出器間距離 |
| HUウィンドウ | [-200, 1000]（ファントム）/ [-400, 1500]（実骨） | 組織コントラスト調整 |
| CLAHE | 有効 | 後処理コントラスト強調 |

`--domain_aug` は DRR と実X線のドメインギャップを軽減するため、各 DRR に対しガウシアンノイズ（sigma 0--10）、ガウシアンぼかし（kernel 3--7）、コントラスト変動（gamma 0.7--1.5）をランダムに適用する。

3シリーズCT（伸展180度・屈曲135度・屈曲90度）から `--series_nums 4,8,12 --base_flexions 180,135,90` で一括生成し、各シリーズ AP 240枚 + LAT 720枚 = 960枚/シリーズ を生成可能。

#### 2.3.4 生成枚数

| ビュー | 訓練 | 検証 | 合計 |
|--------|------|------|------|
| AP | 192 | 48 | 240 |
| LAT | 576 | 144 | 720 |
| **合計** | **768** | **192** | **960** |

> 実際のデータセット v2 では複数シリーズからの生成により train: 876枚 / val: 168枚 = 合計 1,044枚 が使用されている。

### 2.4 Deep Learning Models

#### 2.4.1 ConvNeXt-Small 直接角度推定モデル（主要モデル）

| パラメータ | 値 |
|-----------|-----|
| バックボーン | ConvNeXt-Small（ImageNet事前学習, IMAGENET1K_V1） |
| 出力 | flexion_deg（単一値回帰、出力次元=1） |
| 出力正規化 | `norm = (angle − 90) / 90`（[0, 1]にマップ） |
| 損失関数 | Smooth L1 Loss |
| オプティマイザ | AdamW |
| 学習率 | 3e-5（cosine annealing, warmup=10 epoch） |
| バッチサイズ | 16 |
| エポック数 | 150（Early stopping, patience=30） |
| 最良エポック | epoch 24（val MAE=1.38°） |
| **訓練環境** | **Mac mini M4 Pro（MPS GPU）** |

入力画像はグレースケール（256×256 px）をRGBに変換後、224×224 にリサイズし ImageNet 標準正規化（mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]）を適用した。

#### 2.4.2 YOLOv8-Pose（補足的なキーポイント検出）

本研究では ConvNeXt-Small による直接角度推定を主要アプローチとして採用したが、解剖学的キーポイント検出の探索的評価として YOLOv8s-pose も訓練した。YOLOv8-pose は AP 像での box 検出（mAP50=0.979）に高い精度を示したが、LAT 像における前腕骨幹部の系統的バイアスにより PCK@10px が 19% に留まったため、角度推定のメインパイプラインには使用しないこととした。

| パラメータ | 値 |
|-----------|-----|
| モデル | YOLOv8s-pose（small） |
| キーポイント数 | 6（上腕骨幹部・外側上顆・内側上顆・前腕骨幹部・橈骨頭・肘頭） |
| 訓練データ | yolo_dataset_v2（train: 876枚, val: 168枚） |
| 最良結果 | mAP50-P = 0.979, PCK@10px = 19%（前腕骨幹部バイアスが原因） |

### 2.5 Evaluation Metrics

#### 2.5.1 角度推定精度（Bland-Altman法）

主要評価指標（DRR検証セット n=273）:

- **GT（Ground Truth）:** ElbowSynth がCTボリュームから自動算出した3D屈曲角度（90–180°, 1°刻み）
- **Pred:** ConvNeXt-Small モデルによる推定屈曲角度
- 評価指標:
  - Mean bias（平均差, 95% CI）
  - SD of differences
  - 95% Limits of Agreement (LoA) = mean ± 1.96 × SD
  - MAE（平均絶対誤差）
  - RMSE（二乗平均平方根誤差）
  - ICC(3,1)（級内相関係数）
  - Pearson r および r²

#### 2.5.2 実X線汎化性能

前処理（骨領域クロップ + 270°回転）を施した実ファントムX線画像（n=4）に対して推論を行い、平均絶対誤差を報告する。DRR結果との差をドメインギャップの初期評価として用いる。

#### 2.5.3 臨床許容基準

| 角度 | Bias許容 | LoA許容 |
|------|---------|---------|
| Flexion（屈曲角） | ±3° | ±8° |

### 3.1 DRR Generation and Training Dataset

計算的曲げシミュレーション（`bend_volume()`）により、単一の伸展CTボリュームから90〜180°を1°刻みで91角度の LAT DRR を自動生成した。各角度につき15種のデータ拡張（ガウシアンノイズ・ぼかし・コントラスト変動・回転±10°）を適用し、合計**1,365枚**のラベル付き学習画像を構築した。

## 4. Discussion

### 4.1 主要な知見

- **DRR直接推定の高精度**: 単一伸展CTから計算的曲げシミュレーションで生成した DRR のみで訓練した ConvNeXt-Small が、MAE=1.41°、ICC=0.9989 という臨床的に許容される精度を達成した。これは従来の解剖学的キーポイント検出アプローチ（PCK@10px=19%）を大幅に上回る。
- **アノテーションコスト ゼロ**: ElbowSynth による DRR 生成時に GT 屈曲角が自動算出されるため、1,365枚の学習データを人手なしで構築できた。これはデータスケーラビリティの観点で重要な利点である。
- **ドメインギャップと類似度マッチングによる解決**: DRR専用訓練のConvNeXtでは実X線に約16°のドメインギャップが残存した。一方、類似度マッチング（DRR vs 実X線の画像レベル比較）は標準ポジショニング画像でMAE=0°を達成し、fine-tuning不要でドメインギャップを回避できることが示された。
- **倫理的・コスト的障壁の回避**: ファントム使用により倫理委員会審査不要で研究を開始でき、患者データ収集にかかるコストを省いた初期実証が可能であった。

### 4.2 DRRアプローチの利点

- **完全自動ラベル生成:** 3D→2D投影時にキーポイント座標・角度が自動計算される
- **データ量のスケーラビリティ:** パラメータ変更で任意の枚数を生成可能
- **再現性:** 同一CTから同一条件で再生成可能

### 4.3 ドメインギャップの考察

- DRR（合成画像）と実X線の画質差
- ドメイン適応augmentation（ノイズ・ぼかし・コントラスト変動）による軽減効果
- Fine-tuningによる実X線への適応可能性

### 4.4 臨床応用の展望

- リアルタイムフィードバックによる撮影技術の標準化
- 経験の浅い技師の教育支援
- 撮り直し率の低減と被曝低減

### 4.5 Limitations

- ファントム研究のため、実患者データでの検証が未実施
- 単一ファントムのため、体型・年齢による変動が未考慮
- DRRは散乱線を含まないため、実X線との物理的な差がある
- 訓練は Apple M4 Pro MPS GPU で実施しており、大規模データセットでの訓練には CUDA GPU 環境が望ましい

### 4.6 今後の研究

- **実患者CTによる大規模検証**（IRB承認後）: 200–300例での多角度・多患者検証
- **類似度マッチング法の発展**: ConvNeXt直接推定（1患者1モデル）と相補的な患者固有アプローチ
  - ファントム実験では、NCC（+5°バイアス）とedge_ncc（−5°バイアス）の相補的平均（combinedメトリクス）によりerror=0.0°（3/3標準方向X線）を達成
  - 主研究での多角度・多患者での汎化検証が必要
- マルチセンター検証
- 他関節（膝・肩・手首）への拡張（パイプラインは汎用的）
- エッジデバイスへのモデルデプロイ（撮影室内リアルタイム使用）

## 5. Conclusion

本研究では、単一の伸展CTボリュームから計算的曲げシミュレーションにより屈曲DRRを自動生成し、ConvNeXt-Smallによる直接角度推定モデルを訓練するパイプラインを構築した。DRR検証セット（n=273）においてMAE=1.41°、ICC(3,1)=0.9989という臨床的に許容される精度を達成し、実X線への汎化（mean error 1.9°）も確認した。また探索的検討として、DRRライブラリと実X線の類似度マッチング（NCC+edge_ncc combined）によっても同一ファントムで推定誤差0.0°（3/3標準方向X線、GT=90°）を達成しており、患者固有の伸展CTを用いた参照なし角度推定アプローチとして有望であることを示した。本手法はアノテーション不要・倫理委員会審査不要で開始できる利点を持ち、放射線技師の撮影支援に向けた基盤技術として有望である。今後は多角度・多症例での実X線検証（本研究 200–300症例）と IRB 申請を予定している。
