# 06 精度検証手順（Bland-Altman 解析）

## 目的

AIによる自動計測値と、放射線技師による手動計測値を比較する。
Bland-Altman法で一致性を評価し、臨床使用可否を判断する。

---

## Step 1 : 手動計測データの収集

### 計測対象

モデル訓練に使っていない画像（またはval画像）を使う。
最低20枚、推奨50枚以上。

### 手動計測の方法

施設のPACS/ビューアで以下を計測してCSVに記録する:

| 計測項目 | 計測方法 | 記録単位 |
|---------|---------|---------|
| 外反角（Carrying Angle） | AP像: 上腕骨長軸と前腕骨長軸の角度 | 度（小数点1位） |
| 屈曲角（Flexion） | 側面像: 上腕骨と前腕骨の角度 | 度（小数点1位） |
| 回内外（Pronation/Supination） | AP像: 前腕の捩れ角 | 度（小数点1位） |

### 記録フォーマット（CSV）

```csv
image_id,manual_carrying,ai_carrying,manual_flexion,ai_flexion,manual_pronation,ai_pronation
AP_B_001,10.5,,0.0,,5.2,
AP_C_001,8.2,,0.0,,0.5,
LAT_D_001,9.1,,90.3,,3.1,
```

まず手動計測値だけ入れておく。AI値は次のステップで追記する。

## Step 5 : 結果の解釈

### 見方

```
グラフの読み方:
  横軸 = 手動計測とAI計測の平均値
  縦軸 = AI - 手動計測の差
  中央線（緑）= 平均差（バイアス）
  上下の破線（赤）= ± 1.96SD（一致の限界: LoA）

理想的な結果:
  ・平均差がゼロに近い（バイアスが小さい）
  ・LoAが臨床許容範囲内（外反角なら ±3° 以内が目安）
  ・点が均等に分布している（系統的なズレがない）
```

### 外反角の臨床許容範囲

| 指標 | 目標 | 備考 |
|-----|------|------|
| 平均差（バイアス） | ±1° 以内 | |
| LoA（95%一致限界） | ±3° 以内 | 外反角の臨床的有意差は約5° |
| 相関係数 | r > 0.95 | |

### 屈曲角の臨床許容範囲

| 指標 | 目標 |
|-----|------|
| 平均差 | ±3° 以内 |
| LoA | ±8° 以内 |

## 一括計測スクリプト（batch_analyze.py）

```python
# elbow-train/batch_analyze.py として保存して使う
"""
val画像を一括でAPI解析してCSVに出力するスクリプト

使い方:
  python elbow-train/batch_analyze.py \
    --input data/images/val/ \
    --output validation_output/ai_results.csv
"""
import argparse, csv, os, sys, requests

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--input",  required=True)
    parser.add_argument("--output", required=True)
    parser.add_argument("--api",    default="http://localhost:8000")
    args = parser.parse_args()

    os.makedirs(os.path.dirname(args.output), exist_ok=True)
    images = sorted([f for f in os.listdir(args.input) if f.endswith('.png')])

    with open(args.output, "w", newline="") as f:
        writer = csv.writer(f)
        writer.writerow(["image_id", "ai_carrying", "ai_flexion", "ai_pronation",
                         "ai_varus_valgus", "view_type", "qa_score"])
        for fname in images:
            path = os.path.join(args.input, fname)
            with open(path, "rb") as img:
                r = requests.post(f"{args.api}/api/analyze",
                                  files={"file": (fname, img, "image/png")})
            if r.status_code != 200:
                print(f"SKIP: {fname} ({r.status_code})")
                continue
            d = r.json()
            a = d["landmarks"]["angles"]
            q = d["landmarks"]["qa"]
            writer.writerow([
                os.path.splitext(fname)[0],
                a["carrying_angle"], a["flexion"], a["pronation_sup"],
                a["varus_valgus"], q["view_type"], q["score"]
            ])
            print(f"OK: {fname}  carrying={a['carrying_angle']}  flexion={a['flexion']}")

if __name__ == "__main__":
    main()
```
